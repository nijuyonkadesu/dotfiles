# Dotfiles

Configuration files that I use with Arch

## 1. neovim

All you needed to upscale your vanilla neovim to the Gigachad neovim

- vim be good

### Quick Peek

| Shortcut | Description           |
| -------- | --------------------- |
| ⎵pv      | :Ex                   |
| ⎵gs      | git status            |
| ⎵a       | add to harpoon        |
| Ctrl e   | harpoon quick menu    |
| ⎵pf      | project find          |
| Ctrl p   | git file find         |
| ⎵ps      | project search (grep) |
| ⎵u       | undo tree             |
